Device Integrations allow the addition of device integration data for existing devices in your environment from external SaaS platforms like Microsoft Entra, CrowdStrike Falcon, Microsoft Intune, Sophos Central, etc. Instead of manually correlating separate device records between platforms, users can correlate, review, and audit in one platform.

## Supported Device Integrations

These providers can be used to connect and sync to Tier Zero Code:

- Cloudflare Zero Trust
- CrowdStrike Falcon
- Microsoft Defender for Endpoint
- Microsoft Entra ID
- Microsoft Intune
- Sophos Central
- Qualys
- Tailscale (Coming Soon)

## How to Configure Device Integrations

<Steps>
  <Step title="Navigate to Integration Settings">
    Click on the "Integrations" tab on the left side navbar.
  </Step>
  <Step title="Enable a Device Integration">
    Under the Device Integrations section, select an integration and click the ellipsis (3 dots), then click “Enable Integration” in the dropdown.
  </Step>
  <Step title="Configure Integration">
    Click the ellipsis (3 dots) and click “Configure Integration” in the dropdown. In the modal, enter configuration settings and then click “Save Changes”.
  </Step>
  <Step title="Sync Integration">
    Click the ellipsis (3 dots) and click “Sync Integration” in the dropdown. A notification will appear in the top right corner. The sync will be finished when the status has changed to "Success".
  </Step>
</Steps>
